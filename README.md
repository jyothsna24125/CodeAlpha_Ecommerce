# 🛒 Ecommerce Project

## 📌 About
This is a full-stack ecommerce website built using HTML, CSS, JavaScript, Node.js, and Express.

## 🚀 Features
- Product listing
- Backend API
- Frontend and backend integration
- Simple UI design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## 📂 Project Setup

### Backend
cd backend
npm install
node index.js

### Frontend
Open index.html in browser

## 🌐 API
- GET /api/products
